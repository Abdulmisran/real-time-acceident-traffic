#Traffic Accident Pattern Analysis and Visualization Using Python
The project loads a real road accident dataset from a CSV file and processes it using the Python pandas library. The dataset contains information such as accident date, location, weather condition, road type, severity level, and number of casualties.

After loading the data, the system performs exploratory data analysis to compute totals, distributions, and trends. It then generates interactive visualizations such as hotspot charts, severity pie charts, weather impact graphs, and monthly accident trends.
